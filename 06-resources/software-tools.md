# Software & Tools

**Date:** 2025-11-24

## Game Development

### Selected: Godot Engine
- **Website:** https://godotengine.org
- **License:** Open-source (MIT)
- **Use:** Primary game engine
- **Why:** Free, lightweight, excellent 2D support, user already has it installed
- **Version:** Godot 4.x (latest stable recommended)

## AI/ML Development

### MLGodotKit
- **Website:** https://godotengine.org/asset-library/asset/4060
- **License:** Open-source
- **Use:** Simple ML models in Godot
- **Why:** Easy prototyping, built for Godot

### Python (for ML training)
- **Website:** https://www.python.org
- **Version:** 3.9 or newer
- **Use:** ML model training and integration
- **Why:** Required for TensorFlow/PyTorch training

### PyTorch / TensorFlow
- **PyTorch:** https://pytorch.org
- **TensorFlow:** https://www.tensorflow.org
- **Use:** Primary ML frameworks for training
- **Why:** Industry-standard ML frameworks

### TensorFlow Lite / ONNX Runtime
- **TensorFlow Lite:** https://www.tensorflow.org/lite
- **ONNX Runtime:** https://onnxruntime.ai
- **Use:** Optimized model deployment in Godot
- **Why:** Smaller models, faster inference, better for runtime

### GDExtension
- **Documentation:** https://docs.godotengine.org/en/stable/tutorials/scripting/gdextension/
- **Use:** C++ bindings for ML model integration
- **Why:** High-performance integration with Godot
